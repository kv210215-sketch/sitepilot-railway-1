# QUICK START — SitePilot демо (тільки порядок кліків)

> Одна сторінка. Тримайте перед очима. Повний текст — у `LIVE_DEMO_RUNBOOK.md`.

## Перед стартом (5 хв)
1. `curl https://sitepilot-railway-staging.up.railway.app/health` → має бути `ok`
2. Логін у кабінет: `$DEMO_EMAIL` / `$DEMO_PASSWORD` *(дані не в репо — secure note / Railway / env; див. `.env.example`)*
3. Відкрити вкладки: **Dashboard · Projects · Pages · Publish Queue · Leads · Activity · Public preview · lead-form.html**
4. Термінал готовий (команда набрана, Enter **не** натиснутий): `node demo/solomiya/submit-lead.mjs`

## Порядок демо
| # | Дія | Вкладка / клік |
|---|-----|----------------|
| 1 | Відкриття, 1 фраза | — |

## ❌ НЕ НАТИСКАТИ
- **AI Generate** (може дати 403 — сторінка вже готова, показуємо як «згенеровану»)
- Не покладатись на кнопку форми для створення ліда → лід створює `submit-lead.mjs`
- Жодних деплой / інфраструктурних дій, без видалення проєктів/сторінок/лідів

## 🛟 Якщо збій
- Повільно / лежить → скриншоти `artifacts/shots/01…11`
- Логін злетів → перелогінитись
- Лід не видно → «Оновити» ще раз / перевірити `✓ Лід створено` у терміналі / показати попередній лід
- Preview недоступний → `artifacts/rendered-page.html`
